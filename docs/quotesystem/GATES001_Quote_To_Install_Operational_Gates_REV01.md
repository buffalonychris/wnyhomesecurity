# GATES001 Quote To Install Operational Gates REV01

Status: ACTIVE
Customer-facing: No
Implementation authority: No

## Purpose

This standard defines quote-to-install operational gates for WNYHS quote-system governance.

It is a policy and documentation standard only. It does not authorize payment implementation, Stripe changes, scheduling implementation, customer quote automation, checkout changes, runtime changes, or legal/compliance language for production use.

## Deposit Gate

A 50% deposit is required before scheduling.

No job-specific inventory purchase may occur before the deposit is verified.

No scheduling date may be set until the deposit is verified.

## Final Payment Gate

Final payment is due upon technician arrival on install day unless Chris or Lou explicitly authorize an exception.

Any exception should be documented in the quote record and operator notes before the install date.

## Payment Methods

Accepted payment methods for quote-system governance are:

- Credit card.
- Cashiers check.
- Venmo.
- Cash App.
- Zelle.
- Klarna financing.

## Customer-Facing Quote Language

Customer-facing quote language must politely state:

- A 50% deposit is required before scheduling.
- Job-specific inventory purchase begins only after deposit verification.
- A scheduling date is set only after deposit verification.
- Final payment is due upon technician arrival on install day unless Chris or Lou explicitly approve an exception.
- Accepted payment methods are credit card, cashiers check, Venmo, Cash App, Zelle, and Klarna financing.

## Legal And Compliance Review

Legal and compliance wording requires operator and/or attorney approval before production use.

This document does not approve final New York quote, payment, financing, cancellation, or install-day language for customer use.

## Boundary

This standard defines operational gates only. It does not implement payment verification, Stripe checkout, inventory purchasing, calendar scheduling, quote generation, customer notifications, or any runtime behavior.
