# WNYHS Public Funnel Standards — REV01

Status: LOCKED (governance standard)
Context: CTX-WNYHS-FINISH-LINE-REV01

## 1. Public Funnel Route Roles

- `/home-security` = trust + orientation
- `/packages` = compare starting points
- `/discovery` or `/fit-check` = needs discovery
- `/contact` or `/estimate` = estimate request
- `/support` = help/support
- `/our-work` = proof/trust gallery
- `/about` = company/local trust
- `/privacy` = privacy policy
- `/terms` = service/website terms

## 2. CTA Discipline

- homepage should not be overly salesy
- deeper pages can carry conversion

## 3. Claims Guardrails

Do not use:
- guaranteed protection
- crime-proof/burglary-proof
- police-grade/military-grade
- hack-proof
- always protected
- guaranteed response
- emergency dispatch implications unless supported

## 4. Allowed Language Posture

Prefer language like:
- practical protection
- local-first
- professionally planned
- no mandatory monthly contracts
- equipment options you can own
- clear recommendations
