# WNYHS Brand Asset Standards — REV01

Status: LOCKED (governance standard)
Context: CTX-WNYHS-FINISH-LINE-REV01

## 1. Approved Asset Locations

The following brand assets are approved and locked for WNYHS public use:

- `/public/brand/crest-system/CrestLogo.png`
- `/public/brand/crest-system/IconizedLogo.png`
- `/public/brand/heros/HomePageHero.png`

## 2. Asset Usage Rules

### CrestLogo.png
Use for:
- larger brand moments
- final CTA blocks
- hero/brand emphasis where appropriate

### IconizedLogo.png
Use for:
- nav/header
- footer
- compact icon placements

### HomePageHero.png
Use for:
- homepage hero
- approved WNYHS hero visual reference

## 3. Prohibited Asset Behavior

Do not:
- rename/move/delete brand assets without an explicit task authorization
- insert random generated images into production
- add mockups/screenshots as production assets
- use tactical/military/cyberpunk/fear-based visuals

## 4. Required Image Tone

Brand and hero imagery must maintain:
- premium tone
- local tone
- realistic tone
- calm tone
- bright-enough visibility for clear understanding
- no overly dark treatment
