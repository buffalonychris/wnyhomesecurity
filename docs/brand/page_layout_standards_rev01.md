# WNYHS Page Layout Standards — REV01

Status: LOCKED (governance standard)
Context: CTX-WNYHS-FINISH-LINE-REV01

## 1. Homepage Layout Order (Locked)

The homepage section order is:
1. hero
2. trust strip
3. how it works
4. packages preview
5. our work preview
6. final CTA

## 2. Visual Rhythm Rules

Apply:
- premium restraint
- no app-shell/tiny boxed-page feel
- no excessive nested borders
- calm CTA hierarchy

## 3. Typography Rules

Apply:
- restrained display headline use
- readable body copy
- no overuse of giant headline treatment

## 4. Page Consistency Rules

Public WNYHS pages must:
- feel like one coherent site
- avoid random one-off layouts unless a task explicitly scopes that exception
